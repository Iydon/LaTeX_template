# LaTeX MCM Template

Organized from [GitHub](https://github.com/Liam0205/mcmthesis). Since *mcmthesis* is included by *TeX Live*, I will not attach the code, you can find it from the [website](https://ctan.org/pkg/mcmthesis).

# 解释
- **cfgs**: 配置
- **codes**: 插入代码
- **doc**: *mcmthesis*自带文档
- **examples**: 例子
- **figures**: 图片
- **patch**: Warning修复的补丁
- **sections**: 内容分章节放置
